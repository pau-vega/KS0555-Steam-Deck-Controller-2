            // Positive Y = backward

        }

| Rust test code → Tauri modules | Test code reads source files structurally — no runtime execution of Tauri IPC. Read-only data access. No trust boundary crossed. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-04 | Tampering | validation_test.rs assertions | accept | Tests assert source code patterns. If source changes intentionally, tests must be updated. False positives from formatting changes are acceptable — cost is low (update test). |
| T-10-05 | Information Disclosure | Test output in CI logs | accept | Test output contains source code patterns (function names, UUIDs). UUIDs are public BT24 specification. No credentials or sensitive data. |
| T-10-06 | Repudiation | app.tsx integrity check | mitigate | `git diff --exit-code` detects any modification to app.tsx. CI logs record the failure. Prevents accidental UI changes from being merged without detection. |

## Hardware Test Procedure (non-blocking, per D-10)

**Steam Deck Validation Steps:**

Pre-requisites: AppImage downloaded from CI artifacts, transferred to Steam Deck.

1. **AppImage launch:** `chmod +x RobotController_*.AppImage && ./RobotController_*.AppImage`
   - Expected: Fullscreen window appears with Robot Controller UI (connection status, control pad, status bar)
   - Verify: No crash, no error dialogs

2. **BLE connection (BT24 robot):**
   - Power on the BT24 robot (LED should be blinking)
   - Click "Connect" in the UI
   - Expected: Status shows "Connecting..." then "Connected"
   - Verify: Robot LED goes solid (indicates BLE connection established)

3. **Send commands:**
   - Press F/B/L/R buttons in the UI
   - Expected: Robot moves forward/backward/left/right accordingly
   - Press Stop (S)
   - Expected: Robot stops moving

4. **Gamepad direction:**
   - Verify Steam Deck built-in controller is active
   - Move left stick — verify direction indicator changes to match stick direction
