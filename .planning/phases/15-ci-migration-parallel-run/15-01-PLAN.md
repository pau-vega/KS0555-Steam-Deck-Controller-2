## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-01 | Tampering | Workflow artifact (deb) | mitigate | `actions/upload-artifact@v4` + `download-artifact@v4` with non-overlapping artifact names; artifact integrity is GitHub-managed |
| T-15-02 | Elevation | Per-job permissions | mitigate | D-04 — only `build-flatpak-x64` gets `contents: write`; `build-x64` stays `contents: read`; top-level default is `contents: read` |
| T-15-03 | Information Disclosure | Cache poisoning | mitigate | Cache keys include `hashFiles('**/Cargo.lock')` and `hashFiles('**/pnpm-lock.yaml')` — malicious dependency changes invalidate cache |
| T-15-04 | Denial | Concurrency group | mitigate | D-11 — `cancel-in-progress: true` on `github.ref` prevents duplicate CI runs from consuming runner minutes |
| T-15-05 | Spoofing | Release asset | accept | GitHub Releases API via `GITHUB_TOKEN` is the standard publish path; no additional signing (Flatpak signing is v2.2+ deferred feature SIGN-01) |
</threat_model>
