## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-01 | Spoofing | Gilrs thread | accept | Local gamepad input only, no network exposure. Steam Deck controller verified by name "Steam" (GPAD-05). |
| T-08-02 | Tampering | Event payload | accept | Simple payloads: `{ direction: 'F' }` or `{ name: '...' }`. No sensitive data. Low impact if tampered. |
| T-08-03 | Information Disclosure | Gamepad name | accept | Gamepad name is not sensitive. Only emitted on connect/disconnect events. |
| T-08-04 | Denial of Service | gilrs next_event() | mitigate | Thread runs in background. If gilrs blocks/hangs, it only affects gamepad input, not main app. No explicit handling needed per D-34 (no lifecycle management). |
