# Quick Task Summary: Mac Dev + Low-Friction Steam Deck Install

## What changed

### Mac development support
- `apps/frontend/src-tauri/tauri.conf.json` — `bundle.targets` flipped from `["appimage"]` to `"all"` so Tauri picks the right targets per host (DMG/.app on macOS, AppImage on Linux). Added `bundle.macOS.minimumSystemVersion: "11.0"` (Big Sur — required for CoreBluetooth + Apple Silicon).
- `apps/frontend/src-tauri/Info.plist` (new) — declares `NSBluetoothAlwaysUsageDescription`. Tauri auto-merges it into the bundle. Without this, btleplug silently fails on macOS Big Sur+.
- Verified `cargo check` succeeds on aarch64-apple-darwin (the user's machine). All Rust deps (btleplug, gilrs, tokio, tauri 2.11) build cleanly. Pre-existing `unused_*` warnings unrelated to this change.

### Low-friction Steam Deck install
- `.github/workflows/build.yml` — tagged builds (`v*`) now publish AppImages to a real GitHub Release via `softprops/action-gh-release@v2` (gated on `github.ref_type == 'tag'`). `actions/upload-artifact` retained for manual `workflow_dispatch` runs. AppImages renamed to `RobotController-{x86_64,aarch64}.AppImage` so the installer can hit a stable URL.
- `install-on-steamdeck.sh` (new, repo root) — one-shot installer:
  1. Detects arch (Steam Deck = `x86_64`; `aarch64` supported).
  2. Curls `https://github.com/pau-vega/KS0555-Steam-Deck-Controller-2/releases/latest/download/RobotController-<arch>.AppImage`.
  3. Installs to `~/Applications/RobotController.AppImage` (chmod +x).
  4. Extracts the bundled icon, drops a `.desktop` file in `~/.local/share/applications/`.
  5. Refreshes `update-desktop-database` so the entry appears immediately.
  6. Prints the one remaining manual step (Add as Non-Steam Game).
  Re-running the script upgrades in place.
- `README.md` — rewrote install + dev sections. Removed legacy pre-Tauri instructions: bluetoothctl pairing, `rfcomm bind`, Chrome flatpak override, `WS_PORT`/`SERIAL_PORT` env, the `dev:backend` split. Added a Mac dev section, an *Install on Steam Deck* one-liner, and an updated architecture diagram (`React → Tauri IPC → Rust btleplug/gilrs → BT24`).
